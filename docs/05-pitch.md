# 05 — Pitch do Projeto

## Pitch

Olá, meu nome é Igor e este projeto se chama FinGuard IA, um assistente virtual inteligente criado para apoiar pessoas no controle de gastos pessoais.

O problema que o projeto busca resolver é a dificuldade que muitas pessoas têm em entender para onde o dinheiro está indo e quais gastos podem estar prejudicando o orçamento mensal.

A solução proposta é um agente com Inteligência Artificial Generativa capaz de responder dúvidas sobre consumo, analisar categorias de gastos de forma educativa e sugerir próximos passos simples para economia.

O FinGuard IA utiliza uma base de conhecimento fictícia, composta por transações, perfil financeiro, categorias de gastos e dicas educativas. A partir dessas informações, o agente consegue responder perguntas de forma mais direcionada e útil.

Um dos principais diferenciais do projeto é a preocupação com segurança e confiabilidade. Como o tema financeiro exige cuidado, o agente foi desenhado para não inventar informações, não prometer resultados e informar quando não possui dados suficientes para responder.

Na prática, a pessoa usuária pode fazer perguntas como: "Estou gastando muito com delivery?" ou "Como posso economizar mais este mês?". O agente consulta sua base de conhecimento e responde de forma simples, clara e prática.

Este projeto mostra como a Inteligência Artificial pode ser usada para apoiar decisões do dia a dia, combinando dados organizados, engenharia de prompts e uma aplicação funcional simples.

## Problema

Muitas pessoas possuem dificuldade em acompanhar seus gastos e identificar padrões de consumo que podem afetar o orçamento.

## Solução

Criar um assistente virtual com IA Generativa capaz de responder dúvidas sobre gastos pessoais com base em uma base de conhecimento organizada.

## Valor gerado

O projeto gera valor ao ajudar a pessoa usuária a:

- Entender melhor seus gastos;
- Identificar possíveis excessos;
- Receber orientações simples;
- Tomar decisões financeiras mais conscientes;
- Evitar respostas genéricas ou inventadas.

## Diferenciais

- Uso de base de conhecimento;
- Foco em segurança e anti-alucinação;
- Linguagem simples e educativa;
- Recomendações práticas;
- Protótipo funcional com Python e Streamlit.

## Frase final

O FinGuard IA demonstra como a Inteligência Artificial pode ser aplicada de forma simples, segura e prática para apoiar a educação financeira e a tomada de decisão no dia a dia.
